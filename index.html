🏁 OVERDRIVE

Real Cars. Real Mods. Real Strategy.

About the Game

Overdrive is a competitive, deck-building racing board game that puts real-world performance cars and racing strategy in your hands. Race across iconic track layouts with upgradeable cars, tactical event cards, and a dynamic mod shop system. Outpace, outsmart, and outdrive your rivals!

Core Features

🚗 Real Cars

Choose from accurate, stat-driven cars like the GT86, 370Z, Civic Type R and more — each with unique handling and abilities.

🛠️ Mod System

Upgrade with forced induction, suspension kits, ECU tunes, or track tires. Build a machine that suits your playstyle.

🏎️ Strategic Racing

Racing is turn-based — each decision matters. Use corners, draft zones, and wear tracking to outmaneuver opponents.

📉 Persistent Campaign

Race across 3 tracks, earn CR, repair damage, and build your deck between rounds in a race-weekend style campaign.
